## Javascript class vs object 개념
##### class : 붕어빵을 만들수 있는 틀
- template // 템플릿화 되어있음
- declare once //한 번만 선언
- no data in //데이터가 들어있지 않음

#####  object : class를 이용하여 데이터를 넣어 만듬
- instance of a class //클래스를 이용한 새로운 instance 생성
- created many times //class를 이용하여 많이 만들수 있음
- data in

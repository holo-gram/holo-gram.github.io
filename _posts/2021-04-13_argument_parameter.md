## Argument(인수, 전달인자) Parameter(매개변수)"
```js
function 함수(aaa, bbb){
    return aaa + bbb;
}
// 여기에서 aaa,bbb가 파라미터.
// 함수의 정의에서 사용되는 변수를 파라미터라고 한다.
// 함수와 메서드 입력 변수(Variable) 명
```

```js
contact("parameter", "argument");
// contact 함수를 호출할 때, 입력값 “parameter”와 “argument”는 Argument이다.
// 함수와 메서드의 입력 값(Value)
```
